# sistemaProdutos

Descrição: 
Sistema simples em PHP executado no terminal para gerenciar uma lista de produtos.
Permite listar, buscar, adicionar e remover produtos.

Estruturas de Pastas: 
sistemaProdutos/
│
├── src/
│   ├── data/
│   │   └── produtos.php        # Dados dos produtos
│   │
│   ├── functions/
│   │   └── preco.php           # Funções de cálculo de preços
│   │
│   └── app.php                 # Lógica principal do sistema
│
└── index.php                   # Arquivo inicial que chama o app

Como executar: 
Basta apenas ter o php instalado, e rodar o projeto no terminal. php index.php

Funcionalidades: 
Listar produtos 
Buscar produto por nome
Adicionar produto 
Remover produto
